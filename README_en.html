1 IMESupport for Sublime Text 2

2 Description

In current version of Sublime Text2, there is a problem that can not be displayed IME composition window correctly. This plugin solves this problem.

3 Target platform

This plugin works only on windows.

4 Screenshot

Before installed this plugin. Inptting using IME, but IME composition window is not displayed correct position.

After installed this plugin, IME comosition window is displayed correctly.
